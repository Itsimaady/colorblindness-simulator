# Color Vision Deficiency Simulator

A web-based accessibility tool that helps users understand how images appear to individuals with different types of color vision deficiencies. The application allows users to upload an image and instantly preview simulated versions for Protanopia, Deuteranopia, Tritanopia, and Achromatopsia.

## Features

* Upload and preview images
* Original vs simulated comparison
* Protanopia simulation
* Deuteranopia simulation
* Tritanopia simulation
* Achromatopsia simulation
* Responsive modern interface
* Download simulated images
* Fast browser-based processing using Canvas API

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Canvas API
* FileReader API

## How It Works

The application applies color transformation matrices to uploaded images, simulating how individuals with different forms of color blindness perceive colors. Users can switch between vision modes and compare the original image with the simulated result in real time.
